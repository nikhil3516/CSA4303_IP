CSS Animation Playground

Six classic @keyframes animations — most run automatically, the last one only plays on hover.

Bounce

animation: bounce 1.2s ease-in-out infinite;

Spin

animation: spin 2.5s linear infinite;

Pulse

animation: pulse 1.8s ease-out infinite;

Slide & Fade

animation: slideFade 2.2s ease-in-out infinite;

Color Shift

animation: colorShift 4s ease-in-out infinite;

Wobble (hover to play)

animation-play-state: paused / running;

Animations use @keyframes to define multiple stages, while the animation shorthand controls duration, timing, and repetition.
