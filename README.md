# Welcome to My Website 🌐

A simple, professional single-page website built using **HTML** and **CSS**, featuring a blue-themed design, smooth animations, scrolling ticker text, a styled button, and a responsive grid layout.

## 🔗 Live Preview (Local)

> ⚠️ This link only works on the developer's own machine while running **VS Code Live Server**. It will **not** open for anyone viewing this repository on GitHub.

```
http://127.0.0.1:5500/welcome.html
```

To view it yourself:
1. Clone or download this repository
2. Open the project folder in **VS Code**
3. Install the **Live Server** extension (if not already installed)
4. Right-click `welcome.html` → **Open with Live Server**

## ✨ Features

- Blue-themed background with white and gold accent colors
- Center-aligned hero section with fade-in animation
- CSS-only scrolling ticker text (no `<marquee>` tag)
- Custom-styled button with hover and glow effects
- Responsive grid box layout (3 columns → 1 column on mobile)
- Cambria font family, 14px base size
- Custom-styled scrollbar for a polished feel

## 🛠️ Built With

- HTML5
- CSS3 (Flexbox, Grid, Keyframe Animations)

## 🚀 How to Run

Since this is a static HTML/CSS page, no build tools or server setup are required:

1. Download or clone the repository
   ```
   git clone https://github.com/your-username/your-repo-name.git
   ```
2. Open `welcome.html` directly in any web browser
   **OR**
   Use a live server (recommended) via VS Code / NetBeans for auto-reload while editing
